== Description ==

YITH License Activator is designed to enhance the functionality of your WooCommerce site by providing a simple way to activate YITH's premium plugins. With just a click, you can unlock the full potential of your WooCommerce store, eliminating the need for cumbersome activation processes and intrusive banners.

== Installation ==

1. Upload `yith-license-activator` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the settings under > YITH > License Activation to manage activations.

== Frequently Asked Questions ==

= How do I use this plugin? =
Upload and activate the plugin.

= Is this plugin free? =
Yes, the YITH License Activator is free to use.

= Does this plugin automatically update the YITH plugins? =
No, YITH License Activator is focused on the activation of YITH plugins. It does not handle updates, which are managed through your WordPress dashboard or the individual plugin settings.
