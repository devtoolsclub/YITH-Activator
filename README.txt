== Description ==

YITH License Activator 2.0 is designed to enhance the functionality of your WooCommerce site by providing a seamless way to activate YITH's premium plugins. This updated version introduces better compatibility, enhanced debugging, and slug handling for accurate license application.

With YITH License Activator 2.0, you can:
- Automatically activate YITH's premium plugins without manual intervention.
- Remove intrusive banners and notifications for a cleaner admin interface.
- Save time and streamline your workflow with optimized license management.

== Installation ==

1. Upload `yith-license-activator` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Access the settings under > YITH > License Activation to manage activations.

== Frequently Asked Questions ==

= How do I use this plugin? =
Simply upload and activate the plugin. It automatically detects and activates compatible YITH premium plugins.

= Is this plugin free? =
Yes, YITH License Activator 2.0 is free to use.

= Does this plugin automatically update the YITH plugins? =
No, YITH License Activator focuses on activating YITH plugins. Plugin updates are managed through your WordPress dashboard or individual plugin settings.
